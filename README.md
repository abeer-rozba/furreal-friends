### Description
#### This is a python web application built with django as the final project of my Software Engineering Bootcamp program with General Assembly. Furreal Friends introduces pet owners to like-minded people and advertises pet-related events, fostering a supportive and loving community for animal enthusiasts.

### Technologies Used:
- Python (Django)
- PostgreSQL
- HTML
- CSS

### Getting Started
1. Check out Furreal Friends through this [live website]().
2. Sign up or sign in to your account.
3. Register your pets.
4. Explore events and find like-minded pet parents to connect with.
